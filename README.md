# Stoneburner Interior Designs

Custom website for Stoneburner Interior Designs — SW Florida's premier interior design studio.

## About
- **Designer:** Suzanne Stoneburner
- **Location:** Fort Myers, FL 33908
- **Experience:** 40+ years in Southwest Florida
- **Recognition:** 7× Best of Houzz Award Winner

## Features
- Responsive single-page design
- Auto-sliding hero with project photography
- Scroll-triggered animations & stat counters
- Interactive portfolio grid with lightbox
- Before & After comparison slider
- Testimonial carousel
- Contact form
- **Studio Portal** — internal work hub with:
  - Drag-and-drop vision boards (image upload from desktop)
  - Project scope tracking with checklists
  - Shared links & inspiration
  - Collaborative notes
